# Step 1 Site

This package is ready for GitHub Pages.

## Files
- `index.html` -> landing page
- `style.css` -> shared homepage styling
- `acute-mi.html` -> Acute MI printable guide
- `cv-risk-chest-pain.html` -> CV Risk + Chest Pain printable guide
- `cardio-pathology.html` -> Cardio Pathology printable guide

## Deploy
1. Create a public GitHub repo, for example `step1-site`
2. Upload all files in this folder to the repo root
3. Go to GitHub `Settings -> Pages`
4. Select `Deploy from branch`
5. Choose branch `main` and folder `/(root)`
6. Your site will publish at:

   `https://YOUR-USERNAME.github.io/step1-site/`

## Notion
In Notion, use `/embed` and paste the public GitHub Pages URL.
