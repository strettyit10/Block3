How to deploy this site

  1. Download the zip package from ChatGPT.
  2. Create a new public GitHub repository, for example step1-site.
  3. Upload the contents of the zip directly into the repo root.
  4. In GitHub, go to Settings → Pages.
  5. Choose Deploy from branch, then main and /(root).
  6. After GitHub publishes, your public link will be:
    https://YOUR-USERNAME.github.io/step1-site/
  7. Paste that link into Notion using /embed.
Important: Notion will preserve the visual layout best through Embed. A direct HTML import into Notion will not keep the CSS formatting.
